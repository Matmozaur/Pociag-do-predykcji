# Collector Service

Go service responsible for stage 1 ingestion (raw extraction from PLK API).

## What docs exist today

There is no autogenerated Swagger endpoint exposed by the service runtime.

The source API contract is maintained as OpenAPI in:
- ../../../specs/openapi/collector.yml

## Local endpoints

When collector is running on localhost:8081:
- GET /healthz
- GET /readyz
- GET /api/v1/fetch/status
- POST /api/v1/fetch/dictionaries
- POST /api/v1/fetch/schedules
- POST /api/v1/fetch/operations
- POST /api/v1/fetch/disruptions
- GET /metrics

## Run locally

From repository root:
- make infra-up
- make db-migrate-up

Collector host endpoint:
- http://localhost:8081
