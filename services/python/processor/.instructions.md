# Processor Service Instructions

- Implement only endpoints and schemas defined in specs/openapi/processor.yml.
- Keep all handlers async and fully type annotated.
- Use asyncpg with parameterized SQL only.
- Do not hardcode secrets; use environment variables via pydantic-settings.
- Emit tracing spans for handlers and database calls with service name pociag.processor.
- Use structured JSON logs via structlog.
